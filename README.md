# Ella Ghazarian Portfolio

A React + Vite personal website inspired by the provided reference screenshots and personalized from Ella's resume.

## Edit Your Info

Most content lives in `src/data.js`:

- `name`, `initials`, GitHub, LinkedIn, Devpost, resume, and email links
- home intro text
- project cards and tags
- experience, education, skills, about copy, and interests

## Run Locally

```bash
npm install
npm run dev
```

## Deploy

This project is ready for Vercel. The included `vercel.json` routes deep links back to the app.
